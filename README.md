# laba5

Задача 1: Фильтрация компонентов бинарного файла
Описание задачи
Необходимо заполнить бинарный файл случайными целыми числами и получить в другом файле все компоненты исходного файла, кроме тех, которые кратны заданному числу k.

Решение
Созданы методы:
FillBinaryFileWithRandomNumbers - заполняет бинарный файл заданным количеством случайных чисел в диапазоне 1-100
PrintBinaryFileContents - выводит содержимое бинарного файла в консоль для визуальной проверки
FilterMultiples - считывает исходный файл и создает новый файл, исключая числа, кратные k

Особенности реализации
Используется бинарный формат записи/чтения
Данные сохраняются в формате int
Вывод форматирован по 10 чисел в строке
Включена возможность просмотра исходных и отфильтрованных данных

Задача 2: Работа с матрицами
Описание задачи
Скопировать элементы бинарного файла в квадратную матрицу размером n×n. При недостатке элементов заполнить оставшиеся нулями. Найти столбец, где произведение элементов дальше всего от нуля.

Решение
Созданы методы:

FileToMatrix - преобразует данные из файла в квадратную матрицу
PrintMatrix - выводит матрицу в удобочитаемом формате
FindColumnFarthestFromZero - вычисляет для каждого столбца произведение элементов и находит столбец с максимальным абсолютным значением

Особенности реализации
Обработка недостающих данных (заполнение нулями)
Использование типа long long для произведения для предотвращения переполнения
Нумерация столбцов с 1 для удобства пользователя

Задача 3: Работа со структурами данных
Описание задачи
Файл содержит сведения об игрушках (название, стоимость, возрастные границы). Определить, можно ли подобрать игрушку (кроме мяча), подходящую ребенку трех лет.

Решение
Созданы:
Структура Toy - содержит поля: название, цена, минимальный и максимальный возраст
FillToyFile - заполняет файл случайными данными об игрушках
PrintToyFileContents - выводит список игрушек с форматированием
FindToyForThreeYearOld - проверяет наличие подходящей игрушки

Особенности реализации
Библиотечное сохранение структур с динамическими строками (длина + содержимое)
Проверка всех условий: исключение мяча, проверка возрастного диапазона
Наглядный вывод найденной игрушки при успешном поиске

Задача 4: Обработка текстового файла (одно число в строке)
Описание задачи
Заполнить текстовый файл случайными числами по одному в строке и найти сумму максимального и минимального элементов.

Решение
Созданы методы:
FillTextFileOnePerLine - генерирует файл с числами от -100 до 100
FindSumOfMinMax - находит минимальное и максимальное значения и возвращает их сумму

Особенности реализации
Использование констант INT_MAX и INT_MIN для инициализации
Простой и эффективный алгоритм поиска min/max за один проход

Задача 6: Работа с текстовыми данными
Описание задачи
Заполнить текстовый файл строками (с русскими и английскими буквами) и переписать в другой файл только строки, не содержащие русских букв.

Решение
Созданы методы:
FillTextFileWithRandomText - генерирует файл со смешанными строками
ContainsRussianLetters - проверяет наличие кириллических символов в строке
CopyLinesWithoutRussian - копирует подходящие строки в новый файл

Особенности реализации
Кроссплатформенная обработка кодировок
Учет особенностей UTF-8 при проверке русских букв
Подсчет количества скопированных строк для обратной связи
